The First Chunk

In version 1, the first chunk of ciphertext is reserved for the file name attribute. The decrypted chunk is always 256 bytes long. The encrypted chunk is 276 bytes because the encryption scheme adds 20 bytes.

encryptedChunk = file.read(startOfCiphertext, startOfCiphertext+428)
decryptedChunk = decryptor.decryptChunk(encryptedChunk, false)
